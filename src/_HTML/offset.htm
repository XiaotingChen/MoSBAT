Understanding the offset histogram

 

The offset histogram is a representation of the possible motif alignments. However, instead of showing only one possible alignment, it shows the distribution of possible alignments. In other words, it shows how frequently the two motifs occur at a particular distance from each other in a given set of sequences (e.g. for a set of randomly generated sequences, which is used by default).

The offset histograms are calculated for motif occurrences that are in the same direction and motif occurrences that are at opposite directions separately.
